# Room Window References — Shelter Steam/Desktop

Дата: 2026-06-30
Статус: reference intake folder
Роль: UI/UX + Game Design reference

## Назначение

Эта папка предназначена для скриншотов-референсов, показывающих модель **room window / room grid / inspectable interior**:

- основной экран может оставаться компактным;
- при клике на домик / производственное здание / будку открывается отдельное окно;
- в окне могут быть комнаты, персонажи, декор, обучение, работа, отдых и внутренний театр;
- это не main strip, а detail/inspect layer.

## Почему это важно для Shelter

Для Steam/Desktop Shelter main strip должен оставаться читаемой always-on-top полоской. Но более богатая жизнь собак может происходить во внутренних окнах:

- Dog House / будка открывает жилые комнаты собак;
- Kitchen / Packing / Storage / Laboratory могут открывать производственные комнаты;
- Laboratory может иметь несколько комнат: лаборатория, учебный класс, библиотека;
- собаки могут работать, учиться, читать, отдыхать и помогать друг другу внутри этих окон.

Ключевая формула, зафиксированная пользователем:

> Idle в Shelter означает: «Это жизнь собачек, которая течёт своим чередом, а мы можем за ней наблюдать и чуть-чуть ей управлять».

## Как использовать референсы

Эти изображения не являются прямым art style target. Они нужны как UI/UX и structural reference:

- room-grid / room-window structure;
- click building -> open detail window;
- персонажи живут внутри комнат;
- комнаты могут прокачиваться и украшаться;
- room interior может быть богаче main strip;
- main strip не обязан показывать всю внутреннюю жизнь.

## Boundaries

Allowed to borrow structurally:

- room windows;
- room-grid layout;
- building inspect views;
- interior detail layer;
- dogs/characters visibly doing things in rooms;
- room-specific upgrades and assignments.

Do not borrow:

- dark horror tone;
- combat framing;
- PvP / arena / boss systems;
- aggressive UI pressure;
- gothic violence / sinister atmosphere.
