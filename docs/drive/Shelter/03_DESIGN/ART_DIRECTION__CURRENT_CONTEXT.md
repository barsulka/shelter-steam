## 5. Overlay asset taxonomy

Before generating or assigning any overlay asset, classify it as:

1. **Building** — rare large anchor of the strip.
2. **Utility Prop** — functional object / pause / support prop; must not become a little house.
3. **Dog Action Sprite** — readable dog doing a meaningful action with an object.

Rule:

```text
Buildings are rare anchors. Most visible work should be utility props and dog actions, not a dense decorative village.
```

This rule comes from D-011 cleanup and previous overlay asset pack learnings.
