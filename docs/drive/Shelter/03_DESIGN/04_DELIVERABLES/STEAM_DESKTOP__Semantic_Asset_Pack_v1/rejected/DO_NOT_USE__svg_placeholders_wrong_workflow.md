# DO NOT USE — SVG placeholders created by wrong workflow

Дата: 2026-06-28

Эта заметка фиксирует ошибку процесса: несколько SVG semantic placeholders были созданы через локальный файлсервер вместо правильного workflow:

1. generate via ChatGPT Image / external image generation;
2. review by Game Designer / Art Director;
3. approve or reject;
4. store approved image files in `approved/`.

SVG files created directly through local file server must not be treated as approved visual assets for Codex.

Correct next step: generate image assets with the approved image-generation workflow, then review and store only approved results.
