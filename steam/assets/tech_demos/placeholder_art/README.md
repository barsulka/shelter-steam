# Placeholder Art

Curated local placeholder art for the companion field tech demo.

Raw downloaded/source packs can live in the local ignored `tilesets/` directory.
Only the small files actually used by the demo should be copied here. The
tracked `tilesets/.gdignore` file keeps Godot from importing the raw local
source packs.

Current sources:

- `tx_tileset_ground.png` from `Pixel Art Platformer - Village Props v2.3.1`.
- `tx_village_props.png` from `Pixel Art Platformer - Village Props v2.3.1`.
- `animals/*.png` from `MinifolksForestAnimals/Outline`.

These are temporary development placeholders. They are not the final Shelter art
direction and should be replaced by commissioned production art later.
